# Suitcase E-shop

Modern multi-page e-commerce storefront for luggage and travel accessories built with HTML, SCSS, and TypeScript.

## Live Demo

https://anzbetta.github.io/Suitcase-E-shop/

## Features

* Product catalog with filtering, sorting, search, and pagination
* Product details page with related products
* Shopping cart with persistent state using LocalStorage
* Responsive multi-page layout
* Dynamic product rendering from local JSON data
* Login modal with basic validation
* Cart totals and discount calculations

## Tech Stack

* HTML5
* SCSS (Sass)
* TypeScript
* ESLint
* Stylelint

## Getting Started

Install dependencies:

```bash
npm install
```

Run development watchers:

```bash
npm run dev
```

Build the project:

```bash
npm run build
```

## Available Scripts

```bash
npm run dev
npm run build
npm run lint
```

## Notes

This project was built without frontend frameworks or UI libraries to focus on core frontend architecture, TypeScript modules, responsive layout techniques, and DOM interaction patterns.
